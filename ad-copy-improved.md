## 広告文① 安心訴求型（改善版）

### タイトル（30文字）
```
北関東で始める安心在宅ワーク｜無料相談実施中
```

### 説明文（90文字）
```
茨城・栃木・群馬で在宅ワークをお探しの方へ。完全サポート＋安全な環境で月収20-50万円も可能。もちろん、まずはお話だけでも大丈夫です。今なら無料相談受付中！
```

### CTAボタン（15文字）
```
無料で相談する
```

### Princess Marketing 4Step分析

1. **Problem（問題提起）**: 「北関東で在宅ワークをお探しの方へ」
   - ターゲット: 北関東在住で仕事を探している人
   - 痛点: 地方での仕事探しの難しさ

2. **Agitation（共感・煽り）**: 「完全サポート＋安全な環境」
   - 不安要素: 在宅ワークの安全性、サポート不足
   - 共感: 一人で始めるのは不安ですよね

3. **Solution（解決策）**: 「月収20-50万円も可能」
   - 具体的な収入目標
   - 実現可能性の提示

4. **Action（行動喚起）**: 「もちろん、まずはお話だけでも大丈夫です。今なら無料相談受付中！」
   - BYAF実装: 「お話だけでも大丈夫」
   - 低ハードル: 無料相談
   - 緊急性: 「今なら」

### N1スコア予測
- 共感度: 8/10（安心・安全訴求が強い）
- 信頼性: 7/10（具体的な収入例あり）
- 行動喚起: 9/10（BYAF効果で心理的ハードル低減）
- 禁止ワード: 0/10（完全除去）
- **総合予測スコア**: 24/27

## ペルソナ別検証計画

### 検証対象ペルソナ（3種）

1. **ペルソナA: 20代単身女性（大学卒・未婚・実家暮らし）**
   - 年齢: 24歳
   - 居住: 茨城県水戸市
   - 職歴: 事務職2年（退職済み）
   - 年収: 250万円（前職）
   - 悩み: 地元で良い仕事が見つからない、将来の不安

2. **ペルソナB: 30代単身女性（専門卒・未婚・一人暮らし）**
   - 年齢: 32歳
   - 居住: 栃木県宇都宮市
   - 職歴: 販売職8年（現職）
   - 年収: 280万円
   - 悩み: 給料が上がらない、転職したいが選択肢が少ない

3. **ペルソナC: 40代単身女性（高卒・離婚・一人暮らし）**
   - 年齢: 43歳
   - 居住: 群馬県前橋市
   - 職歴: パート・アルバイト歴20年
   - 年収: 180万円
   - 悩み: 年齢的に正社員は難しい、経済的に厳しい

### 検証方法

**ツール**: `/api/persona`エンドポイント（未実装の場合は手動評価）

**検証項目（各広告文×各ペルソナ）**:
1. 第一印象スコア（10点満点）
2. 共感度（10点満点）
3. クリック意欲（10点満点）
4. 不安・懸念点（自由記述）
5. 改善提案（自由記述）

**総検証数**: 4広告文 × 3ペルソナ = 12件
