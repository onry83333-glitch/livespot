## Critical（使えないレベル）— 8件

### C1. [ダッシュボード] データ鮮度が不明 → 意思決定できない
KPIカード（売上30d, チャット1h, DM7d）に「いつ時点の数字か」の表示がない。マネージャーは古いデータで判断するリスク。
→ **改善**: 各KPIカードに「最終更新: 14:45 JST」タイムスタンプ追加

### C2. [ダッシュボード] 配信中キャストの状態が見えない
ログイン後、どのキャストが今ライブ中かわからない。/castsに行くまで不明。
→ **改善**: 「配信中キャスト」セクションを最上部に追加（視聴者数付き）

### C3. [サイドバー] DM・分析・ユーザーへのナビリンクが存在しない
サイドバーに /dm, /analytics, /users, /alerts へのリンクがない。主要機能が発見できない。
→ **改善**: サイドバーに「DM管理」「売上分析」「ユーザー」セクション追加

### C4. [DM] キャンペーンCVR（効果測定）が見えない
DM送信後、成功/失敗のステータスは見えるが「そのDMで売上が上がったか」は不明。dm_effectiveness RPCは実装済みだがUIに未接続。
→ **改善**: バッチステータスカードの下に「効果測定」パネル追加

### C5. [分析] DM効果が常に0%表示
Analytics画面のDM conversion rateが0%固定（実装スタブ）。マネージャーは「DM効果なし」と誤解。
→ **改善**: 未実装なら「Coming soon」バナー表示。実装済みRPCに接続するか非表示に

### C6. [SPY] キャスト別監視状態が不明
複数キャスト監視時、どれが実際にデータ受信中かわからない。サイレント切断に気づけない。
→ **改善**: キャスト別ステータスアイコン（🟢受信中/🟡一時停止/🔴切断）

### C7. [セッション] AI分析レポート生成ボタンが存在しない
generateReport()は450行以上実装済みだが、UIにボタンがない。機能が完全に隠れている。
→ **改善**: 各セッションカードに「AIレポート生成」ボタン追加

### C8. [ダッシュボード] "GO LIVE"ボタンが非機能
サイドバーに目立つ緑のボタンがあるがonClickハンドラなし。クリックしても何も起きず混乱。
→ **改善**: 機能実装するか完全に削除

## High（毎日ストレスを感じる）— 18件

### H1. [ダッシュボード] Whale一覧からアクション不可
Top15 Whale表示だが、クリックしてもDM送信もプロフィール確認もできない。「見えるけど動けない」。
→ **改善**: ユーザー名クリックで /users/[username] 遷移 + 「DM送信」ボタン追加

### H2. [ダッシュボード] 離脱リスクセクションが無効化されたまま表示
churn riskが0件表示だがバックエンド未デプロイのため実際は無効。「離脱リスクなし」と誤認。
→ **改善**: 機能無効時はセクション非表示 or 「準備中」バナー

### H3. [ダッシュボード] デモデータ挿入ボタンが本番に表示
空の状態で「デモデータ挿入」ボタンが表示。本番DBを汚染するリスク。
→ **改善**: 開発モード限定にするか確認ダイアログ必須

### H4. [ダッシュボード] サーバー状態が常に「最適化済み」
ハードコードされた偽ステータス。実際の障害時も緑表示。
→ **改善**: pipeline_statusテーブルに接続して実データ表示

### H5. [キャスト詳細/分析] S1-S10セグメントの意味がわからない
セグメント名はあるが定義の凡例なし。「S1=最上位? 最下位?」が不明。
→ **改善**: 折りたたみ可能な凡例カード追加（2軸: コイン額×最終課金日）

### H6. [キャスト詳細/分析] セグメント→DM遷移でコンテキスト喪失
「N名にDM送信」ボタンでDMタブに移動するが、どのセグメントから来たか表示されない。
→ **改善**: DMフォームヘッダーに選択セグメント名表示 + campaign名に自動入力

### H7. [キャスト詳細/売上] SPYとコインAPIの2つの数字が混在
「今週売上」がSPYベースとAPIベースで2つ表示。どちらが正しいか不明。
→ **改善**: 「リアルタイム推定」vs「確定データ」のラベル明示 + 差分率表示

### H8. [キャスト詳細/分析] リテンションステータスの定義が曖昧
🟢Active/🟡AtRisk/🔴Churned/🆕Newの閾値（7日/14日）が画面に表示されない。
→ **改善**: ステータス横に「7日以内にチップ」等の定義を明示

### H9. [DM] お礼DM機能がAPIオフラインで無言で故障
v2 API未デプロイで「候補なし」と表示。オフラインなのか本当にゼロなのか区別不能。
→ **改善**: API不通時は「⚠️ API接続不可 — 手動モードで動作中」表示

### H10. [DM] 画像アップロードUIが非機能
ドラッグ&ドロップエリアがあるがonClickハンドラなし。何も起きない。
→ **改善**: 機能実装するかUI要素を削除

### H11. [SPY] フィルター設定がセッション間で保持されない
msg_typeフィルタが毎回リセット。ページ離脱→再訪で全型表示に戻る。
→ **改善**: localStorage にフィルタ状態を保存

### H12. [設定] Chrome拡張の接続状態が見えない
拡張が接続中/切断中/未インストールかの表示がゼロ。DM送信可否がわからない。
→ **改善**: 「Chrome拡張: 接続中 ✓」バッジ or 最終通信時刻表示

### H13. [設定] セキュリティタブがモックデータ
「保護スコア」「セキュリティログ」が全てハードコード。保存しても何も永続化しない。
→ **改善**: 未実装機能は「Coming soon」表示 or 非表示

### H14. [ユーザー] ユーザーカードからDM送信できない
Whaleユーザーを見つけてもDM送信ボタンなし。手動でDMページに移動して名前入力が必要。
→ **改善**: 各ユーザーカードに「📧 DM送信」ボタン追加

### H15. [ユーザー] ページネーションなし
10,000ユーザーを一括レンダリング。ブラウザがラグる。
→ **改善**: 50件/ページのページネーション or 無限スクロール

### H16. [キャスト一覧] ライブ状態カラムがない
キャスト一覧テーブルに「今配信中か」の情報がない。各キャストをクリックしないとわからない。
→ **改善**: テーブルに🟢/⚫ ライブステータス列追加

### H17. [全体] tk/¥ 単位の不統一
ページによって「12.5K tk」「¥96,250」「$xxx」が混在。どの数字を信じるか混乱。
→ **改善**: 全ページで「nnnn tk (¥xxxx)」を併記。設定のcoin_rateを各ページフッターに表示

### H18. [ダッシュボード] プライマリアクションが不明
ログイン後、3つのクイックアクションが等重量で表示。「最初に何すべきか」の優先順位なし。
→ **改善**: データに基づく「推奨アクション」カード（例: 「5名の離脱WhaleにDM送信」）

## ユーザージャーニー検証

### ジャーニーA: 「配信前にDMを送りたい」
- **現状**: ダッシュボード → /casts クリック → キャスト選択 → 分析タブ → セグメント展開 → DM送信ボタン → DMタブに移動 → 確認モーダル → 送信 = **5画面 8クリック**
- **詰まりポイント**: サイドバーに/dmリンクがない(C3)。セグメント→DM遷移でコンテキスト喪失(H6)
- **改善後**: ダッシュボード推奨アクション → セグメント自動選択 → DM確認 → 送信 = **2画面 4クリック**

### ジャーニーB: 「昨日の配信結果を確認したい」
- **現状**: ダッシュボード → /casts → キャスト選択 → 売上タブ → SPY/API数字の違いに混乱 = **4画面 5クリック + 混乱**
- **詰まりポイント**: 2つの売上数字の意味不明(H7)。AIレポートボタンなし(C7)
- **改善後**: ダッシュボード → キャスト → セッション → AIレポート自動生成 = **3画面 3クリック**

### ジャーニーC: 「DM送信後の効果を確認したい」
- **現状**: /dm → バッチステータス確認 → 完了 → **CVR確認の導線がない** → 行き止まり
- **詰まりポイント**: CVRデータ未表示(C4)。dm_effectiveness RPCは存在するがUI未接続
- **改善後**: /dm → バッチ → 効果測定パネル → 来訪率/課金率/売上貢献 = **1画面 2クリック**

### ジャーニーD: 「新キャストを追加して使い始めたい」
- **現状**: /settings → キャスト追加 → Chrome拡張の状態不明(H12) → コイン同期方法不明 → 行き止まり
- **詰まりポイント**: オンボーディングガイドなし。Chrome拡張連携の手順説明なし
- **改善後**: /settings → キャスト追加 → ステップバイステップガイド → 拡張設定 → 初回同期 = **1画面 ガイド付き**
