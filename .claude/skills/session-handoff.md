# セッション引き継ぎ手順

## トリガー
セッション終了時、または手動で `/session-handoff` を実行

## 手順
1. 現セッションの作業内容をサマリーする
2. 未完了タスクがあれば、状態を記録:
   - 何をしていたか
   - どこまで進んだか
   - 次に何をすべきか
   - 注意すべきこと
3. Notionタスクページのメモ欄に引き継ぎ情報を追記
4. 進行中タスクのステータスは変更しない（次のセッションが引き継ぐ）

## フォーマット
```
### セッション引き継ぎ（YYYY/MM/DD HH:MM）
- 作業内容: {サマリー}
- 進捗: {完了率}
- 残タスク: {箇条書き}
- 注意点: {次のセッションへの申し送り}
```
